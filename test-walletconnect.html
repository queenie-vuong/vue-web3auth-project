WalletConnect Debug Test

Open the browser console and navigate to http://localhost:5173

Test Steps:

  1. Open browser console (F12)
  2. Click "Connect with WalletConnect" button
  3. Watch console logs for state updates
  4. Connect your wallet
  5. Check if Web3Auth component hides
  6. Check if UserInfo component shows

Debug Commands (run in console):

// Test WalletConnect connection
await window.testWalletConnect()

// Check store state
const store = document.querySelector('#app').__vueApp__.config.globalProperties.$pinia._s.get('wallet')
console.log('Wallet store state:', {
  address: store.address,
  connectionType: store.connectionType,
  isConnected: store.isConnected,
  isWalletConnectConnected: store.isWalletConnectConnected,
  isWeb3AuthConnected: store.isWeb3AuthConnected
})
